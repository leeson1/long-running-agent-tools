# long-running-agent-tools - Claude Code 工作规范

## 一、提交规范

向 leeson1 的仓库提交代码时：

- **author 必须是** `leeson1 <leeson1@users.noreply.github.com>`
- **禁止** 添加 `Co-Authored-By` 行

配置方式：

```bash
git commit --author="leeson1 <leeson1@users.noreply.github.com>" -m "..."
```

或在仓库级别设置：

```bash
git config user.name "leeson1"
git config user.email "leeson1@users.noreply.github.com"
```
